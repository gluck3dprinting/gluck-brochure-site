# GLUCK Sales Brochure 2026

산업용 SLA 3D프린팅 기반 고정밀 양산 제조 파트너 **글룩(GLUCK)** 의 인터랙티브 영업 소개자료 저장소입니다.
이 저장소는 **보관·서빙(GitHub Pages) 역할**을 하며, 소스 편집과 빌드는 사내 시스템에서 이뤄집니다.

## 📌 주소 체계

| 구분 | 주소 | 설명 |
|---|---|---|
| **대외 공식** | `https://glucklab.com/brochure/` | 홈페이지 도메인 연결 주소 (래퍼 → 최신 공유본, 기기별 자동 분기) |
| **최신 공유본** | [`/preview/`](https://gluck3dprinting.github.io/gluck-brochure-site/preview/) | 항상 최신 — 사내 [배포] 버튼으로 갱신됨 |
| 최신 · 랜딩 | [`/preview/landing.html`](https://gluck3dprinting.github.io/gluck-brochure-site/preview/landing.html) | 세로 스크롤 버전 |
| 최신 · 모바일 | [`/preview/mobile.html`](https://gluck3dprinting.github.io/gluck-brochure-site/preview/mobile.html) | 모바일 가로보기 뷰어 |
| 구 공식본(보존) | [`/`](https://gluck3dprinting.github.io/gluck-brochure-site/) | 기존에 공유된 링크 유지용 — 갱신하지 않음 |
| 명함 QR 진입점 | [`/m`](https://gluck3dprinting.github.io/gluck-brochure-site/m) | 모바일 뷰어로 고정 리다이렉트 |

## 📁 저장소 구조

```
├─ index.html / landing.html / mobile.html   ← 구 공식본 (보존)
├─ factory-2025.mp4                          ← 팩토리 필름 (웹 압축본)
├─ m/                                        ← 명함 QR 고정 진입점
└─ preview/                                  ← ★ 항상 최신 공유본 (자동 갱신 대상)
   ├─ index.html      매거진(1600×900 페이지 넘김, 40p)
   ├─ landing.html    세로 스크롤 랜딩
   ├─ mobile.html     모바일 가로보기 뷰어
   └─ factory-2025.mp4
```

각 HTML은 **단일 파일 완결형**입니다 — 이미지(base64)·폰트(SUIT)·커버 로티 애니메이션이 모두 내장되어
파일 하나만 있으면 오프라인·보안망 환경에서도 동일하게 열립니다. (영상만 별도 mp4 참조)

## 🔄 제작 · 배포 파이프라인 (사내)

```
src.html (클린 소스, 사내 관리)
   │  python build.py / build_landing.py / build_mobile.py
   ▼
index·landing·mobile.html (이미지·폰트·로티 인라인 빌드)
   │
   ├─ 사내 뷰어(8765) ─ 저장 시 4초 내 자동 새로고침
   ├─ 디자인 모드(8770) ─ 팀 실시간 협업 편집 → 저장 시 자동 재빌드
   └─ [🌐 배포] 버튼 / python deploy.py → 이 저장소 /preview/ 푸시 → 1~2분 내 반영
```

### 디자인 모드 주요 기능
- 피그마식 편집: 클릭 선택 · 드래그 이동(스마트 가이드 스냅) · 8핸들 리사이즈 · 방향키 미세이동
- 텍스트: 더블클릭 수정, 부분 서식(볼드·색·크기), **새 텍스트 추가 · 복제(Ctrl+D) · 복사/붙여넣기(Ctrl+C/V)**
- 이미지: 클릭 교체 · 업로드(자동 웹 최적화: 1800px 리사이즈, 사진형 PNG는 고품질 JPG 변환, 원본 백업)
- 협업: 접속자 프레즌스(커서·선택 표시), 동시 저장 충돌 보호, 동료 저장 감지 자동 갱신
- SVG 공정 도식 텍스트 편집, 사례 팝업 편집, 페이지 배경 편집
- **[🌐 배포]**: 저장된 최신 빌드를 이 저장소 `/preview/`로 원클릭 게시

## 🕓 최근 주요 업데이트

- **커버 리디자인** — 신규 2막 로티 애니메이션(단일 베드 빌드 → 9베드 팜, 호버 시 2막 재생) + 브랜드 딥톤 컬러
- **양산사례 갤러리 2장 체제** — 산업군 1p + 예술·산업 1p, 양산 베드 실사 22컷 신규 선별·색보정
- **P21 수량 스펙트럼** — 1ea/800ea/6,000ea+/15,000ea+ 구간별 실물 이미지 카드 4종
- **판단 기준 표 5축 개편** — 형상 · 일정 · 설계 · 리스크 · 비용
- **트랙 구조 서사** — 브릿지 양산(트랙 A) + 금형 대체 불가 형상(트랙 B) 2트랙 메시지
- **AI 재구성 이미지 적용** — 보안 유지가 필요한 사례 컷은 AI 재구성 이미지 + 고지 문구로 대체 (진행 중)
- **뷰어·배포 자동화** — 저장 시 사내 뷰어 자동 새로고침, 원클릭 외부 배포

상세 버전 이력은 사내 `versions/VERSIONS.md`에서 관리합니다.
